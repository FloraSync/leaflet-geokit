Marker icon customization

This harness keeps tiles local, seeds stable point fixtures, and puts the default Leaflet marker next to a custom SVG marker so Playwright can catch broken asset loads, sizing regressions, and silent fallbacks to the stock icon.

What This Page Proves

Loaded markers and draw markers stay on the same icon contract.

  • The default reference map keeps the bundled Leaflet marker.
  • The custom reference map seeds a larger SVG icon through marker attributes.
  • The custom map keeps `draw-marker` active so Playwright can place a second marker and verify the draw tool uses the same asset.
